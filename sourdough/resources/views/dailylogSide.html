The left-hand custom view is rendered using the following files:
  • sourdough/resources/views/dailylog.html
  • sourdough/resources/views/dailylog.view.xml
  • sourdough/resources/views/dailylog.webpart.xml
The scripts and styles may be found in the directory:
  • sourdough/resources/web/dailylog
For examples on how to load custom CSS and JS files, as well as how to import third party dependencies, see:
  • dailylog.view.xml
For examples on how to make LabKey API calls within your scripts, see:
  • sourdough/resources/web/dailylog/script.js
